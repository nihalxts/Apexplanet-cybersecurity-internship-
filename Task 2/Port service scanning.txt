# Port and Service Scanning

## Objective

Port scanning is used to identify open ports and running services on a target system. This helps security professionals understand possible entry points in a network.

## Tool Used

Nmap (Network Mapper)

## TCP Scan

A TCP scan checks which TCP ports are open on the target system.

Example command:

```
nmap -sS <target-ip>
```

This scan is also known as a SYN scan and is commonly used for stealth scanning.

## UDP Scan

UDP scanning checks UDP ports that may be open on a system.

Example command:

```
nmap -sU <target-ip>
```

UDP scans are usually slower but useful for detecting services like DNS and SNMP.

## Service Version Detection

This scan identifies the version of services running on open ports.

Example command:

```
nmap -sV <target-ip>
```

Example output may show services such as:

* FTP
* HTTP
* SSH
* MySQL

## Operating System Detection

Nmap can attempt to detect the operating system running on the target machine.

Example command:

```
nmap -O <target-ip>
```

## Conclusion

Port and service scanning helps identify exposed services in a network which could potentially contain vulnerabilities.
